此项目旨在增强「独立中文博客」的互动。

## 收录标准
- 拥有独立域名的个人博客
- 不收录：纯技术、纯网络、转载资源、网赚、主机资讯等类别的博客

非硬性但可参考的一般标准：
1. 博客创立时间 2 年以上
2. 每年发布博文不少于 5 篇
